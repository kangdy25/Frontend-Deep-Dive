# 🚀 Frontend Deep Dive

> **Frontend Engineering Beyond Frameworks**
>
> 단순히 라이브러리와 프레임워크를 사용하는 방법을 넘어,
> 프론트엔드 엔지니어링의 기반이 되는 컴퓨터 과학(CS), 소프트웨어 공학, 디자인 패턴, 그리고 웹 플랫폼의 동작 원리를 깊이 있게 탐구하는 저장소입니다.
>
> 프레임워크는 변하지만, 원리는 남습니다.

---

## 🎯 Purpose

현대 프론트엔드 개발은 단순한 UI 구현을 넘어 복잡한 상태 관리, 성능 최적화, 아키텍처 설계, 그리고 유지보수 가능한 시스템 구축을 요구합니다.
이 저장소는 이러한 문제를 해결하기 위한 근본적인 원리와 엔지니어링 사고방식을 학습하고 정리하기 위해 만들어졌습니다.

## ⚙️ Algorithm

프론트엔드 애플리케이션에서도 다양한 알고리즘과 자료구조가 사용됩니다.
본 섹션에서는 성능 최적화와 문제 해결 능력의 기반이 되는 알고리즘을 직접 구현하고 분석합니다.

---

## 🏗️ Design Pattern

확장 가능하고 유지보수하기 쉬운 소프트웨어를 설계하기 위한 다양한 디자인 패턴과 설계 원칙을 연구합니다.
단순한 패턴 암기가 아닌, 실제 프론트엔드 아키텍처에 어떻게 적용되는지에 집중합니다.

---

## 🛠️ Software Engineering

좋은 소프트웨어는 단순히 동작하는 코드가 아니라 지속적으로 유지보수할 수 있는 코드입니다.
의존성 관리, 결합도, 응집도, 아키텍처 설계 등 소프트웨어 공학 전반을 탐구합니다.

## 📈 Engineering Philosophy

### Never Assume, Verify

동작을 암기하지 않고 직접 검증합니다.

### Understand Before Using

사용하기 전에 원리를 이해합니다.

### Build From First Principles

추상화 뒤에 숨겨진 본질을 탐구합니다.

### Theory Meets Practice

이론과 실제 구현을 연결하여 학습합니다.
